This app logs workout sessions with a text-based interface, with easy to "hit" inputs, solving common frustrations with typical notes apps.

A notebook and pen would provide better UX, but they're not easy to carry in a gym, so this webapp allows storing logs on your phone, and exporting the workout elsewhere, as text.

## Tech stack
- HTML, CSS and Vanilla JS with plenty of web components
- shoelace.style helps with widgets and theming
